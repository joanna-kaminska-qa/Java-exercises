## Description

This project includes several small exercises implemented in Java:

- **Arrays.java** – exercises related to arrays (sorting, searching, manipulation)
- **Rozne.java** – miscellaneous Java exercises
- **wyszukiwarkaKsiazek.java** – a simple book search program demonstrating logic and data handling

The purpose of these exercises is to practice:

- Java syntax and basics
- Object-oriented programming
- Algorithms and data structures
- Console input/output

##Getting Started

### Requirements

- Java 21
- IntelliJ IDEA (or any Java IDE)

### Running the Exercises

1. Open the project folder in your IDE.
2. Compile and run any of the `.java` files directly from the IDE.
3. Modify code as needed for practice and experimentation.
